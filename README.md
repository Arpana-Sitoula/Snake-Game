# Not Only Snake Game - Computer Graphics Project

This project was developed as part of a semester module for Computer Graphics. The initial codebase and project structure are based on the class activity template provided by the professor.

## Architecture

The project follows the **MVC (Model-View-Controller)** design pattern to ensure a clean separation of concerns between game logic, rendering, and user input.

## Game Modes

The application features two distinct modes managed by a high-level site manager:

1.  **Exploration Mode**: A 3D environment where the user can freely roam and explore the room.
2.  **Game Mode**: A 2D gaming experience (Snake) where the user can engage in gameplay.

## Interaction

The transition between exploration and gaming is handled via an **Interaction Zone** located in front of the screen. When the user enters this zone, they are prompted and can switch into the Game Mode.
